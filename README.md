# WebTyo Example Charts

This repository contains example HTML files that demonstrate how to embed or display charts sourced from a Google Sheets spreadsheet.

## Files

- `index.html`
- `index1.html`
- `index2.html`

## Purpose

These files are example templates for showing charts whose data is loaded from Google Docs / Google Sheets. They are intended as illustrative examples rather than production pages.

## Usage

Open any of the HTML files in a browser to view the chart example. The files show how a chart can be rendered using spreadsheet data from Google Sheets.

## Notes

- The source data in these examples is expected to come from a Google Sheets spreadsheet.
- You may need to update the spreadsheet URL or data access settings if you want to adapt the examples to your own sheet.
